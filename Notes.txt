-- Startup Cash Test
----------------------------------------------------------------------
1) Enable test crash env variable
$env:SCRYSCREEN_TEST_STARTUP_CRASH=1
dotnet run --project .\ScryScreen.App\ScryScreen.App.csproj -c Debug
2) Disable test crash env variable
Remove-Item Env:SCRYSCREEN_TEST_STARTUP_CRASH -ErrorAction SilentlyContinue

Or with an absolute path:
powershell -NoProfile -ExecutionPolicy Bypass -File "D:\Dev\AI\ScryScreen\publish.ps1" -Runtime win-x64 -Configuration Release -Zip
